About

Googuardiaan is a CMD playground.

It looks like a retro command prompt mixed with a modern game launcher, but everything is focused on learning how programs think.

Retro terminal vibe

The UI is inspired by classic command prompts, neon arcade machines, and modern portfolio sites.

Built for learners

Perfect for students and hobbyists who want to understand variables, branches, and I/O without setting up heavy tools.

Ethical only

Labs are designed to teach analysis and debugging, not to evade security or monitoring systems.

Docs

Commands & usage.

Overview

Googuardiaan runs as a CMD‑style program. You type commands and options, and it simulates how a simple tool would respond. 

C:\User> googuardiaan --lab echo
C:\Lab\Echo> help

Commands

  • help – Show all available commands.
  • run <lab> – Start a learning lab.
  • explain – Show step‑by‑step breakdown.
  • notes – Open the in‑app notebook.

Ethical Use

Googuardiaan exists to teach how software works, not how to bypass security, filters, or monitoring systems.

Every lab focuses on debugging, reasoning, and safe behavior. Use it to get better at programming and analysis, not to break rules.
